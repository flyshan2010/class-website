# 🎓 學生系統工作流（五庫一流程）

> 2026-07-10 建立。核心理念：老師只做「記錄」與「匯入」，其餘由系統自動匯流成報告。

## 資料流總覽

```
👥 學生基本資料（名冊）─────────────────┐
   座號/姓名/查詢碼/職務/週薪/職務說明     │ 個資只填一次，
   ↑ 幹部指派（關於我們頁公開，姓名去識別化）│ 全系統 relation 引用
                                        ▼
📝 班經與學習紀錄庫 ──/class-log── 一句話記錄課堂與生活表現（含金幣影響自動計算）
                                        │
💯 學生成績庫 ──/class-grades── 成績單上傳即匯入（定期評量五科/名次/進步獎）
                                        │
🏦 班級銀行帳本 ──/class-bank── 週結：發薪（名冊職務週薪）→ 結算獎懲（紀錄庫金幣影響）
                                → 進步獎（成績庫進步獎 +20）→ 利息
                                        │
                                        ▼
📊 學生學習報告 ──/class-report── 每週自動彙整：
   五向度分數（紀錄庫）＋考試成績等第（成績庫）＋各科建議（紀錄＋成績）
   ＋亮點＋家長協助建議 → 老師檢查 → 勾發布 → 班網加密查詢
```

## 老師的例行操作（全部一句話）

| 時機 | 動作 | 範例 |
|---|---|---|
| 平時隨手 | /class-log 記錄 | 「座號5 數學小考粗心 -1」 |
| 考完試 | /class-grades 匯入 | 上傳成績單 PDF「匯入成績」 |
| 週五放學 | /class-bank 週結 | 「class-bank 週結」 |
| 週末 | /class-report 生成 | 「產生本週學習報告」→ 檢查 → 勾發布 |
| 學期初 | 名冊維護 | Notion 填職務/週薪/職務說明（幹部自動上關於我們頁） |

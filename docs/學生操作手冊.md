## 二、要用「座號＋查詢碼」的（個人區）

查詢碼由老師私下發給每位同學（和家長），**每個人不一樣，請不要告訴別人**。

### 📊 學習報告（家長）
1. 班網「學習報告」→ 輸入**座號**與**查詢碼**→ 解密開啟。
2. 內容：五向度雷達、各科狀態與建議、亮點、目標、家長協助建議；考試週多一張成績級距圖；有作品的週會顯示「🎨 本週作品」。
3. 可按「🖨️ 列印」（一張 A4）或「🕶️ 去識別化」（姓名變 王○明，適合留存展示）。

### 🏦 小小銀行（同學）
1. 班網「小小銀行」→ 輸入**座號**與**查詢碼**→「📖 打開存摺」。
2. 存摺會顯示目前餘額與最近 30 筆交易（薪水、獎勵金、利息、消費…）。

### 🛒 我要兌換（同學）——用崑山幣換商店的東西
1. 打開存摺後往下看商店櫥窗：**買得起又有庫存**的商品會出現「🛒 我要兌換」按鈕。
2. 錢不夠的商品會顯示「🪙 還差 N 幣」——那就是你的存錢目標！
3. 按下「我要兌換」→ 確認 → 送出申請。**這時候還不會扣錢**，要等老師確認。
4. 老師核可後才會扣崑山幣（存摺會多一筆「🛍️ 消費」）；如果被駁回，老師會寫原因，錢不會扣。
5. 小提醒：一次最多 3 筆申請在等老師處理；送出前想一想「這是需要還是想要？」

## 三、注意事項

- 🔒 查詢碼＝你的個人鑰匙，不要外流；忘記了請找老師。
- 存摺與報告內容只給自己和家長看，請勿轉傳截圖。
- 頁面資料每天自動更新三次；老師處理完兌換後也會手動更新，稍等一下再重新整理即可。

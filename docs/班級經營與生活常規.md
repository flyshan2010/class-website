## 二、一日作息與常規對照表

| 在校時間起迄 | 班規理念 | 班級常規（S.O.P.） | 場域期待行為 | 違反後果（邏輯後果） |
| :--- | :--- | :--- | :--- | :--- |
| **07:20–07:40**<br>早晨入班與準備 | 認真負責<br>尊重他人 | **入班三步驟**：掛包包 ➔ 放水壺 ➔ 依科目交作業至指定籃。<br>**定心預備**：回座位整理桌面或抄聯絡簿，0–1 級音量，不追逐嬉戲。 | 【教室】輕聲進教室，作業排好、桌面乾淨，道早安後安靜回座。<br>【走廊/廁所】不逗留交談；進教室前先上廁所洗手。 | **補齊責任**：作業未交或亂放者，下課整理補交。<br>**倒退重做**：進門用擠的或用跑的，回門口重新用走的進來。 |
| **07:40–08:00**<br>環境晨掃 | 認真負責<br>注意安全 | **晨掃 S.O.P.**：07:40 提示即取掃具 ➔ 完成個人分工 ➔ 07:55 前收尾。<br>**工具歸位**：掃具「哪裡拿哪裡放」，抹布擰乾掛平，洗手台清空。 | 【教室/外掃區】負責區掃乾淨，見垃圾主動撿，不拿掃具嬉鬧。<br>【走廊/洗手台】排隊有序，甩乾水份不弄濕地板。 | **重新打掃/修復**：不認真者下課補滿或重掃指定區。<br>**暫停使用掃具**：不當使用掃具玩鬧者，暫停使用並宣導安全。 |

### 導師實務提醒
1. **晨掃銜接**：07:40 播固定「打掃音樂」，學生聽到即自動開工；07:55 音樂停＝收尾歸位。
2. **晨間看板**：黑板右側每日標示 08:00 是「全校朝會」或「班級晨讀」，入班一眼看清。
3. **午休分段**：12:30–12:40 潔牙與如廁期，12:40 正式關燈趴下，避免手忙腳亂。

```

        原理＝代價設計 v3（自然後果 ＞ 補償型 ＞ 限制型）
第二線｜崑山幣回饋訊號（公式不變）
        金幣 ＝ 正負向 × 程度(1–3) × 5；亮點 × 10；扣到 0 為止
        ⚠️ 扣幣是「回饋」，必搭配第一線的真實代價，不能只扣錢了事
第三線｜重複/對抗型 → 輔導個案
        兩週同類負向紀錄 ≥3 或單筆程度 3 → /class-counsel 開案
        （ABC 行為觀察 → 功能假設 → 代價設計，見 class-counsel skill）
```

## 四、增強物制度（2026-07-24 依 CW-FIT 研習資料建置，取國小階段）

### 四類增強物與班級對應

| 類別 | 性質 | 班級做法 |
|---|---|---|
| 社會性增強物 | 免費、即時、最優先使用 | 口頭具體讚美、聯絡簿寫給家長的好話、全班每人給他一句讚美、好寶寶卡蓋章 |
| 活動增強物 | 體驗類，商店兌換 | 桌遊同樂、黑板創作、音樂點播（見 §商店品項） |
| 特權增強物 | 殊榮類，商店兌換 | 老師椅子體驗、提早五分鐘下課、免打掃、班級小主人（見 §商店品項） |
| 實物增強物 | 最後手段、延宕滿足 | 商店小物區＋期末夢幻餐點券 |

使用原則（研習重點）：**社會性 ＞ 活動/特權 ＞ 實物**；能用免費的讚美就不動用商店；
商店品項的教育目標是「儲蓄與延宕滿足」，不是「用獎品換行為」。

### 刻意排除品項（專業理由，避免日後重提）

| 排除項 | 理由 |
|---|---|
| 免處罰券 | 牴觸邏輯後果原則——行為的真實代價不可購買豁免（📜 頁抽獎池既有「豁免金牌」屬里程碑獎勵，性質不同，保留） |

| 指定同學幫自己做事 | 變相使喚的人際風險 |
| 不用排路隊回家 | 中年級放學安全考量 |
| 手機類、不用寫數學 | 國高中階段項目／損及學習權 |
| 辦同學會、換一節自由活動 | 全班性增強物，由老師主導舉辦（如全班存款總額達標禮），不進個人商店 |

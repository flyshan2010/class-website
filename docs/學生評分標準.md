# 🎯 學生學習報告・評分項目與標準（給學生的努力方向）

> 目的：讓每位同學看得懂報告上四個等第（內容評量／作業成績／上課參與／生活常規）
> 是怎麼來的、要怎麼做才會進步。標準全部對齊系統實際計分規則
> （/class-log 記錄 → /class-report 換算），老師答得出「為什麼是這個等第」。

等第一律用五級：**特優 ＞ 優 ＞ 甲 ＞ 乙 ＞ 丙**。
**甲＝基準值**（符合期待、穩定表現；沒有特別紀錄時預設為甲）；乙、丙代表要加油。

---

## 一、內容評量（平時學習的評量表現）

- **平常週資料來源**：紀錄庫「評量」類紀錄（隨堂小考、聽寫、形成性評量等）
  ＋ 學期成績庫的「平時」分（平時 50% 的部分）。
- **期考週資料來源**：改用學期成績庫三科平均（平時 50% ＋ 期考 50%）。
- **成績組成**：學期成績 ＝ 平時 50%（日常表現 20% ＋ 平時考 10% ＋ 作業 20%）＋ 期考 50%。
- **等第對照（期考週依分數換算）**：

| 等第 | 三科平均 |
|---|---|
| 特優 | 90 分以上 |
| 優 | 80–89 分 |
| 甲 | 70–79 分（基準） |
| 乙 | 60–69 分 |
| 丙 | 60 分以下 |

- **怎麼努力**：平時的每一次小考、聽寫都算數 → 平常認真準備，不必等到期考才衝刺。
  進步分數 ≥ 10 或拿到「進步獎」，會自動寫進報告「學生亮點」並加發 20 崑山幣。

---

## 二、作業成績

- **資料來源**：紀錄庫「作業」類紀錄（準時／遲交／缺交／訂正／用心）的正負向累積。

| 等第 | 表現 |
|---|---|
| 特優 | 全部準時、內容用心，主動加做或協助同學 |
| 優 | 準時繳交、完整用心，會主動訂正 |
| 甲 | 大致準時完整，偶爾需提醒（基準／無紀錄預設） |
| 乙 | 常需催交、品質普通、少訂正 |
| 丙 | 常缺交或遲交、應付了事、未訂正 |

- **怎麼努力**：聯絡簿記下作業→回家先寫完→睡前檢查書包；被退的作業訂正可加分。

---

## 三、上課參與

- **資料來源**：紀錄庫「課堂表現」類紀錄（發言、專心、討論、秩序）的正負向累積。

| 等第 | 表現 |
|---|---|
| 特優 | 積極發言、帶動討論，能協助同組同學學習 |
| 優 | 主動舉手、認真參與討論 |
| 甲 | 專心聽講、被點到能回答（基準／無紀錄預設） |
| 乙 | 較少參與、偶爾分心 |
| 丙 | 常分心、講話、需多次提醒，影響上課 |

- **怎麼努力**：每節至少舉手一次、分組時負責一個小任務、有疑問就發問。

---

## 四、生活常規

- **資料來源**：紀錄庫「生活指導」＋「生活技能」類紀錄（秩序、整潔、物品管理、
  值日打掃、自理）的正負向累積。

| 等第 | 表現 |
|---|---|
| 特優 | 常規優良、主動維護環境，能提醒帶動同學 |
| 優 | 整潔守規、打掃值日盡責 |
| 甲 | 常規平穩、無特別事件（基準／無紀錄預設） |
| 乙 | 偶爾違規、東西亂放，需提醒 |
| 丙 | 常違反班規、打掃隨便，需反覆提醒 |

- **怎麼努力**：下課先收桌面、排隊安靜、輪到值日認真做完再走。
- **提醒**：負向紀錄除了扣崑山幣，還會對應「同領域代價」
  （例如：弄亂→自己整理好；打掃沒做完→補做），做到即可修復。

---

## 五、五向度雷達圖（國語／數學／社會／人際互動／生活技能）

- 每個向度從 **3 分（中間值）** 起算，滿分 5。
- 每一則紀錄依公式加減：**正負向（＋/－）× 程度（1–3）× 0.5**。
  例：一則「＋2」的紀錄 → +1.0 分；一則「−1」→ −0.5 分；中性不計。
- 該週該向度沒有任何紀錄 → 沿用上週分數（沒有上週 → 3 分）。
- 對照：國／數／社 ← 該科課堂、作業、評量紀錄；人際互動 ← 人際紀錄；
  生活技能 ← 生活技能＋生活指導紀錄。

**一句話**：想讓雷達圖變大，就是平時多累積「正向、程度高」的好表現紀錄。

---

## 六、金幣（崑山幣）連動

- 每一則正負向紀錄自動換算金幣：**金幣 ＝ ±1 × 程度 × 5**；亮點類 ＝ 程度 × 10。
- 每週五「週結」統一入帳：發薪（職務週薪）→ 結算獎懲 → 進步獎 → 利息（餘額 5%，上限 20）。
- 所以「上課參與／作業／生活常規／內容評量」表現好，不只等第漂亮，錢包也會變厚。

> 本標準由班級系統自動計分，老師僅微調；每學期可依班級狀況調整級距。

# 班網老師操作手冊

## 一、日常怎麼用（90% 的情況只需要這段）

所有內容都在 Notion 的「**🏫 班級經營中心**」頁面維護：
https://app.notion.com/p/394b1f9d7e4581dcb65ec41e587d55ba

| 想做的事 | 去哪裡 | 重點 |
|---|---|---|
| 寫今日聯絡簿 | 📒 聯絡簿 | **115學年度上課日已全部預排好**（六日與國定假日已排除），找到當天那筆填作業，**勾「發布」** |
| 發公告 | 📣 公告 | 「來源」選班級或學校；圖片拖進「圖片」欄會顯示在班網；勾「發布」上線、「置頂」排最前 |

**多行欄位（作業、提醒事項等）一行寫一項**，班網會自動變成條列。

### 日課表怎麼改
在 Notion「**🕐 日課表**」資料庫改，一列就是一節課（星期＋節次＋科目＋教師＋教室）：
- **換科目／改老師名字／改教室**：直接改那一列的欄位。
- **調課**：改「星期」或「節次」即可。
- **停開某節**：取消勾「顯示」（或刪除該列），該格就會變空白。
- 晨掃、早自修、朝會、午休與週三／週五半天等固定時段由網站自動產生，不用建。
- 節次時間、科目底色與下方說明文字寫在 `scripts/sync-notion.mjs` 的 `SCHEDULE_META`，要改再進 repo。
改完等自動同步（07/16/20 時）或到 Actions 手動 Run workflow 立即生效。

### 學生學習報告怎麼發（每週）

在 Notion「**📊 學生學習報告**」資料庫填寫，一列＝一位學生一週：

1. 每週開新列最快的方式：打開 Claude Code 說「**產生本週學習報告**」→ AI 自動以上週為底幫全班建好本週列（分數等第沿用、建議與亮點清空），老師只微調有變化的部分。
2. 「期間」填週次（例：四上第2週(9/7-9/11)），與班級週報一致。
3. 「頭貼」欄可放學生照片（小圖即可，400KB 內）；沒放就沿用該生前一週的頭貼，都沒有則顯示通用圖示。
4. 五向度分數（1～5 畫雷達圖）＋四個等第（考試/作業/上課參與/生活常規）＋各科狀態建議＋亮點/目標/家長協助建議。
5. 勾「**發布**」→ 等自動同步（或一鍵更新）→ 家長到班網「📊 學習報告」頁輸入**座號＋查詢碼**查看；多週會自動變成報告頁的週次切換。

家長端功能：**🖨️ 列印報告**（一張 A4 直式）、**🕶️ 去識別化**（姓名變「王○明」、座號與頭貼隱藏，適合列印留存或展示）。

🔒 隱私設計：報告與頭貼在同步時就用該生查詢碼**加密**，班網上沒有明文；查詢碼建議每位學生不同，私下發給該家長；沒發布的列完全不會上傳。
資料庫裡有一列「30 王小明（示範用，可刪）」查詢碼 1234，可先體驗效果再刪除。

### 首頁「近期行事」
需要 Google 日曆開啟公開：Google 日曆 → 設定 → 該日曆 →「存取權限」→ 勾「公開這個日曆」。開啟後班網會自動抓近期事件；沒開啟也不影響行事曆頁的嵌入畫面。

## 二、班網什麼時候更新？

- 自動：每天台灣時間 **07:00、16:00、20:00** 同步一次。
- 立即更新（電腦）：到 GitHub repo → **Actions** → 「同步 Notion 資料到班網」→ **Run workflow**。
- 立即更新（手機一鍵）：見下方「手機一鍵更新班網」。

### 手機一鍵更新班網（iOS 捷徑，設定一次終身受用）

**步驟 1：建一把專用鑰匙（GitHub Token）**
1. 手機或電腦開 https://github.com/settings/personal-access-tokens/new
2. Token name 填 `class-website-sync`；Expiration 選 **No expiration**（或一年，到期再換）。
3. Repository access 選 **Only select repositories** → 勾 `class-website`。
4. Permissions → Repository permissions → **Contents** 設為 **Read and write**，其他不動。
5. 按 Generate token，**複製整串 `github_pat_` 開頭的文字**（只會顯示一次）。

**步驟 2：建 iOS 捷徑**
1. 打開「捷徑」App → 右上 **＋** 新增捷徑，命名「更新班網」。
2. 加入動作：搜尋「**取得 URL 內容**」（Get Contents of URL），設定：
   - URL：`https://api.github.com/repos/flyshan2010/class-website/dispatches`
   - 方法：**POST**
   - 標頭（Headers）加兩條：
     - `Authorization` → `Bearer 貼上剛才的token`
     - `Accept` → `application/vnd.github+json`
   - 請求本文（Request Body）：選 **JSON**，加一個欄位 `event_type` = `sync-now`
3. （可選）再加一個「顯示通知」動作，內容填「班網同步中，約 1 分鐘後生效 ✅」。
4. 完成！可把捷徑加到主畫面或桌面小工具，**一點就同步**。

⚠️ Token 等同這個 repo 的鑰匙，只放在自己手機的捷徑裡，不要貼給別人或貼進聊天群。

## 三、一週三報（每週五 5 分鐘）

1. 在「📰 班級週報」新增一筆，填五欄位：學習重點／班級活動／學生亮點／下週提醒／家長配合事項。
2. 狀態改為「**待產文案**」。
3. 打開 Claude Code 說「**幫我產生本週週報三端文案**」→ AI 會把「班網版／LINE版／FB版」寫回 Notion。
4. 檢查文案 → 複製 LINE 版貼班群、FB 版貼社團 → 勾「LINE已發」「FB已發」。
5. 狀態改「**已發布**」→ 班網週報頁自動出現完整版。

## 四、相簿照片流程

1. Google Drive 建活動資料夾（放在「班級活動照片」母資料夾下）。
2. 資料夾權限設「**知道連結的任何人：檢視者**」。
3. 照片丟進去。
4. Notion「🖼️ 相簿索引」新增一筆：活動名稱＋日期＋資料夾連結＋勾「顯示」。
5. 下次同步後班網就有縮圖牆。

⚠️ 照片上網前請確認學生肖像權同意書；避免單一學生特寫＋全名同框。

## 五、初次設定（只做一次）

### 1. 建 Notion integration（讓 GitHub 能讀資料）
1. 到 https://www.notion.so/my-integrations → 「New integration」。
2. 名稱填 `class-website-sync`，Workspace 選自己的，Capabilities 只需 **Read content**。
3. 複製 **Internal Integration Secret**（`ntn_` 開頭）。
4. 回到 Notion「🏫 班級經營中心」頁面 → 右上「…」→「Connections」→ 加入 `class-website-sync`。

### 2. 建 Google Drive API key（相簿用，可先跳過）
1. https://console.cloud.google.com/ → 建專案 → 啟用「Google Drive API」。
2. 「憑證」→ 建立 API 金鑰 →（建議）限制只能呼叫 Drive API。

### 3. 把金鑰放進 GitHub
GitHub repo → Settings → Secrets and variables → Actions → New repository secret：
- `NOTION_TOKEN` = Notion integration secret
- `DRIVE_API_KEY` = Drive API 金鑰（沒有也能跑，只是相簿沒縮圖）

### 4. 修改班級基本資料
校名、班級、學年度、班級口號、「關於我們」都在 Notion「⚙️ 網站設定與關於我們」維護。
日課表在 Notion「🕐 日課表」資料庫維護。Google 日曆嵌入網址、選單項目與模組顏色在 `data/site-config.json`（技術設定，一般不用動）。

## 六、學期回顧頁

班網的「🎓 學期回顧」頁**全自動**：把已發布的週報（亮點、活動、照片）與相簿彙整成統計數字＋亮點牆＋活動剪影＋每週足跡，不用維護，期末打開就是現成的班級成果展。
